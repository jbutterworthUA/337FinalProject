Movie Catalog & Watchlist Web Application

This is a Node.js web application built using Express.
Users can create accounts, log in, browse a movie catalog, add movies to their watchlist, mark movies as watched, and view or assign ratings out of 5 stars.
Each user has a personalized catalog stored in the databases/user_catalogs folder.

1. Required Packages:
- npm install express

2. Start the server:
- node server.js

The server will run at: http://localhost:3000
